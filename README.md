# 🏋️ Gym Members Exercise Tracking Analysis

A data science project that analyzes gym members’ exercise habits, workout performance, and fitness-related patterns using Python. The project focuses on understanding how workout type, session duration, gender, age, water intake, and experience level affect calories burned and overall exercise behavior.

---

## 📌 Project Overview

This project was developed as part of a Data Science course project. The main goal is to identify the most effective workout strategies for achieving fitness goals using exploratory data analysis and hypothesis testing.

The analysis uses a gym members exercise tracking dataset and applies data visualization, statistical summaries, correlation analysis, and hypothesis testing to extract meaningful insights.

---

## 🎯 Main Objective

The major objective of this project is:

> To identify the most effective workout strategies for achieving specific fitness goals.

To achieve this, the project explores several important fitness-related questions:

- How are calories burned affected by workout session duration?
- Which workout type contributes more to calorie burning?
- Does water intake relate to calories burned?
- Does water intake differ according to experience level?
- Which workout types require more water intake?
- Which workout type has the highest session duration?
- Is there a relationship between age and calories burned?
- Which gender burns more calories?
- How does workout frequency differ by gender and workout type?
- Does Cardio burn more calories than HIIT?
